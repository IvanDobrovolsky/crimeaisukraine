# Social Media Platforms — Crimea Location Representation

**Research date:** 2026-03-30
**Methodology:** Web research, platform URL analysis, secondary source review

---

## 1. Instagram (Meta)

**Rating:** ❌ Incorrect / ⚠️ Mixed (multiple conflicting location tags coexist)

**Current state:** Instagram maintains **multiple, contradictory location tags** for Crimean cities:

| Location Tag | URL | Country Label |
|-------------|-----|---------------|
| Crimea, Sevastopol | instagram.com/explore/locations/960461116/ | Not specified |
| Crimea, Ukraine | instagram.com/explore/locations/759916349/ | **Ukraine** ✅ |
| Russia, Crimea, Yalta | instagram.com/explore/locations/955620049/ | **Russia** ❌ |
| Crimea (generic) | instagram.com/explore/locations/258865686/ | Not specified |

**Assessment:** Instagram allows users to tag the same Crimean cities under BOTH "Ukraine" and "Russia" location labels. The platform has not consolidated or corrected these conflicting geographic entries. Users can freely choose to tag "Russia, Crimea, Yalta" which effectively normalizes the Russian claim.

**Action needed:** Meta should remove or relabel location tags that place Crimean cities under "Russia."

---

## 2. TikTok

**Rating:** ✅ Correct (regional classification fixed)

**Key change (2022):** TikTok moved Ukraine from a shared "region" with Russia to the **European region**, following advocacy from Ukraine's Ministry of Digital Transformation.

**Context:** Prior to the change, Ukraine and Russia were grouped together in TikTok's internal regional classification system. Ukraine's Deputy Minister for IT Development confirmed the separation.

**Crimea-specific policy:** No specific TikTok policy documentation was found regarding how Crimean cities are classified in location tags. The regional change applies to Ukraine as a country, but whether individual Crimean city tags are under "Ukraine" or "Russia" requires direct platform testing.

**Source:** [IMI: TikTok moves Ukraine from shared "region" with Russia](https://imi.org.ua/en/news/tiktok-moves-ukraine-from-shared-region-with-russia-ministry-of-digital-transformation-i46927)

## 3. Facebook (Meta)

**Rating:** ⚠️ Insufficient data for current state

**Known:** Facebook uses location data from its Places database, which is shared with Instagram. The same conflicting location entries that exist on Instagram likely exist on Facebook.

**No specific policy documentation** was found regarding Facebook's treatment of Crimean city locations in 2024-2026.

**Action needed:** Direct verification of Facebook location/check-in options for Simferopol, Sevastopol, and Yalta.

---

## 4. X (formerly Twitter)

**Rating:** ⚠️ Insufficient data for current state

**Known:** X allows users to set a profile location (free text) and optionally geotag individual posts. The platform does not enforce geographic boundary validation on user-entered locations.

**No specific policy documentation** was found regarding X/Twitter's treatment of Crimean city locations in its structured location database.

**2025 update:** X introduced a feature showing user country on profiles based on phone number/registration data. How this applies to users in Crimea (who may have Russian or Ukrainian phone numbers) is unknown.

**Source:** [InfoHubFacts: X profile location country feature](https://www.infohubfacts.com/technology/x-profile-location-country-feature)

## Key Finding

The most significant issue is on **Instagram/Meta platforms**, where both "Crimea, Ukraine" and "Russia, Crimea, [city]" location tags coexist. This allows and implicitly endorses dual sovereignty claims. Given Instagram's massive reach, this is a high-priority target for correction.

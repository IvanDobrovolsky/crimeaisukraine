Proof: world-atlas@2/countries-50m.json assigns Crimea to Russia

This map loads the exact same TopoJSON file used by every D3/Observable world choropleth: world-atlas@2/countries-50m.json. Russia (country ID 643) is red. Ukraine (804) is blue. Crimea renders as red because it falls inside Russia's polygon.

Data source chain: Natural Earth shapefile → topojson/world-atlas npm → this map. The world-atlas README says: "a convenient redistribution of Natural Earth's vector data."

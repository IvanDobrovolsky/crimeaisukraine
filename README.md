# 🇺🇦 Crimea Is Ukraine

**How do maps, data libraries, streaming platforms, travel services, and internet infrastructure classify Crimea?**

We audited **121 digital platforms** across 12 categories. Crimea is internationally recognized as Ukrainian territory (UN GA Resolution 68/262, 100-11 vote), illegally occupied by Russia since 2014.

## Key Findings

**1.** Natural Earth assigns `SOVEREIGNT=Russia` to Crimea. This cascades to 30.4M weekly npm downloads across D3, Plotly, Leaflet, ECharts. Only Highcharts overrides it. 33 GitHub issues demanding the change — all ignored.

**2.** After 2022, consumer platforms changed (Apple Maps, Netflix, TikTok). Developer infrastructure did not (Natural Earth, Plotly, IANA tzdata, D3).

**3.** Legal/registration services classify Crimea as Ukraine (MaxMind, Cloudflare, GeoNames, OSM). Operational services classify it as Russia (phone routing, timezones, postal codes).

**4.** Weather services: 10/10 correct. If weather apps can get it right, so can everyone else.

**5.** Reuters, AP, BBC use "annexed" consistently. RT and Sputnik frame Crimea as Russian territory.

**6.** Multiple political figures across Europe have made explicit sovereignty endorsements — documented with dates and sources in the research paper.

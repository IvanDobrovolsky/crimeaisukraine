# Crimea Is Ukraine

Investigative research documenting how digital platforms represent Crimea's sovereignty.

Crimea is internationally recognized as Ukrainian territory, illegally occupied by Russia since 2014. This project systematically checks maps, travel platforms, media outlets, and tech services for incorrect representations.

## Structure

```
docs/          → Findings by category (maps, travel, media, tech)
data/          → Raw data (JSON/CSV)
scripts/       → Collection/analysis scripts
SPEC.md        → Full project specification
CLAUDE.md      → Agent instructions
```
